理解JavaScript的立即调用函数表达式(IIFE)

五月 8, 2017 编程语言

首先这是js的一种函数调用写法，叫立即执行函数表达式(IIFE，即immediately-invoked function expression)。顾名思义IIFE可以让你的函数立即得到执行(废话)。

一般来说，IIFE有以下几种用途：

  1. 创建只使用一次的函数，并立即执行它。
  2. 创建闭包，保存状态，隔离作用域。
  3. 作为独立模块存在(例子如jQuery)，防止命名冲突，命名空间注入(模块解耦)。
